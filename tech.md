# Gijiroku21 Technology Stack
Gijiroku21は、ローカルで高精度な議事録作成を実現するために、以下の技術スタックとアーキテクチャを採用しています。
## フロントエンド（デスクトップアプリUI）
- **フレームワーク**: Tauri (Rustバックエンド + Reactフロントエンド)
- **UIライブラリ**: React + TypeScript + Radix UIコンポーネント
- **状態管理**: React Context API + カスタムフック
- **スタイリング**: Tailwind CSS
- **ビルドツール**: Vite
