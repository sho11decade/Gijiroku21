以下は、**「ローカル動作・NPU活用・プライバシー重視」を前提とした議事録アプリ**において、
既存競合との差別化ポイントを最大限に活かすための **UI提案**です。
対象は *個人ユーザー／中小企業／自治会*、主利用シーンは *会議* を想定しています。

---

## 1. UI全体コンセプト

### コンセプト名

**「安心が可視化される議事録UI」**

### 基本方針

* **クラウドを使っていないことが“一目で分かる”**
* **AIが“勝手にやっている感”を排除**
* **ITリテラシーが高くないユーザーでも使える**
* **会議の流れを妨げない**

---

## 2. メイン画面構成（会議中UI）

### ① 会議ダッシュボード（1画面完結）

```
┌───────────────────────────┐
│ ● 会議名：〇〇自治会 定例会          │
│ ● 状態：録音中（ローカル処理）       │
│ ● AI処理：NPU使用中 🟢               │
├───────────────────────────┤
│ [リアルタイム文字起こし表示]         │
│ ・発言者A：〜〜〜                    │
│ ・発言者B：〜〜〜                    │
│ （確信度が低い箇所は薄色表示）       │
├───────────────────────────┤
│ [重要マーク] [決定事項] [要確認]     │
├───────────────────────────┤
│ 🔒 この音声・文字は外部送信されません │
└───────────────────────────┘
```

### 差別化ポイント

* **「NPU使用中」「ローカル処理中」を常時表示**
* プライバシーをUIで“宣言”する設計
* クラウド前提UIとの差別化が明確

---

## 3. プライバシー可視化UI（重要差別化）

### ② プライバシーステータスバー

画面下部または上部に常時表示：

```
🔒 処理場所：このPC
🧠 AI：ローカルLLM
📡 通信：なし
```

* 通信が発生する場合（例：ユーザーが要約をエクスポート）

  * **黄色 or 赤で警告表示**
* 自治会・企業向けに**説明責任を果たせるUI**

---

## 4. 会議後UI（議事録整理）

### ③ 議事録エディタ（人が主役）

#### 特徴

* WordやGoogle Docsに近い感覚
* AIは「提案」しかしない

```
[AI要約案] ← 右側に表示
--------------------------------
本文（人が編集）
```

* AI生成部分は**色付き or 枠付き**
* 「AI生成部分を全削除」ボタンあり

→
**「AIに書かされた」感覚を排除**
特に自治会・役所系ユーザーに強く刺さる

---

## 5. NPU活用を活かすUI表現

### ④ パフォーマンス可視化（控えめ）

設定画面や詳細表示で：

* 処理時間（例：音声→テキスト 0.8秒）
* CPU / NPU 使用割合（簡易グラフ）

→
エンジニア・技術好き層への訴求
「軽い・速い・ローカル」の納得感

---

## 6. 初回起動・説明UI（重要）

### ⑤ 初回オンボーディング（3画面）

**1枚目：**

> このアプリは
> **音声を外部に送信しません**

**2枚目：**

> AIはこのPCの中だけで動きます
> （インターネット不要）

**3枚目：**

> 会議に集中してください
> 議事録は後から整えられます

→
競合（クラウド系議事録AI）との**明確な思想の違い**

---

## 7. 設定画面UI（中小企業・自治会向け）

### ⑥ 設定は「日本向け・実務向け」

* 難解なAI設定は隠す
* ON/OFF中心

```
☑ 発言者分離を使う
☑ 要点抽出を使う
☐ 自動要約（会議後）
```

＋
**「この設定は何のため？」説明文を必ず表示**

---

## 8. 競合UIとの差別化まとめ

| 項目     | 競合（クラウドAI） | 提案UI     |
| ------ | ---------- | -------- |
| プライバシー | 設定画面に記載    | 常時表示     |
| AIの立場  | 主役         | 補助       |
| 動作環境   | クラウド前提     | 完全ローカル   |
| 安心感    | 抽象的        | 視覚的・具体的  |
| 想定ユーザー | IT企業       | 一般組織・自治会 |
